# side_quests/ — the same rigor pointed at OTHER venues

After the 5-minute market walled, the method (rigor gate + second-mind refutation) was pointed at
adjacent hunting grounds. Every one of these also resolved to a clean kill — which is itself the
strongest replication of the method. Some scripts read the sibling whale-tracker project's database
or external data; they are research harnesses, not part of the collector pipeline.
